Desc: This code is meant to grab values from AHT2x and display them
on the LCD display

Disclaimer: My project used; a LCD w/o I2C, AHT2x with I2C, and uses ESP32 Dev Module



       LCD PIN-OUTS
___________________________
	
rs -> GPIO4
en -> GPIO16
d4 -> GPIO17
d5 -> GPIO5
d6 -> GPIO18
d7 -> GPIO19

*GND & VCC is not mentioned on this pinout sheet*


      AHT2x PIN-OUTS
___________________________
	
SCL -> GPIO22
SDA -> GPIO21

*GND & VCC is not mentioned on this pinout sheet*
